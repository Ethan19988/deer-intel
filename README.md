# deer-intel

AI-powered hunting intelligence platform.

The active application lives in [`deer-intel-v2`](./deer-intel-v2) — a Next.js
app with an interactive Leaflet map, properties, trail cameras, stands, hunt
log, live weather, and an AI Scout. See
[`deer-intel-v2/README.md`](./deer-intel-v2/README.md) to get started.

The original static prototype has been archived in [`legacy/`](./legacy) for
reference and is no longer maintained.
