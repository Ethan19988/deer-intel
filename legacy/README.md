# Legacy prototype (archived)

This folder holds the original static, single-file prototype of Deer Intel:

- `index.html`
- `app.js`
- `styles.css`

It was a browser-only mockup (localStorage, hardcoded demo login) used to
sketch the idea. It has been **superseded by the Next.js app in
[`../deer-intel-v2`](../deer-intel-v2)**, which is the actively developed
product (Leaflet map, properties, cameras, stands, hunt log, live weather,
AI Scout, optional cloud sync).

Kept for reference only — no longer maintained. New work goes in
`deer-intel-v2`.
